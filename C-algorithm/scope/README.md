# 学习函数作用域和存储状态
#### 测试结果
全局变量一个存储区域

全局静态变量一个单独存储区域

局部变量一个存储区域	**stack**

代码段一个存储区域

动态变量一个区域(malloc/new)，没有测试！**heap**

#### 结论
**一般的说法就是静态区域，没有区别对待，但是gcc测试结果全局静态和全局变量的地址明显是不连续的！(C/C++都是如此)**

不同函数的静态变量和全局静态变量是放在一起的，虽然函数的局部静态变量的作用域，限制在函数里面共享，但是使用指针还是很容易读取数据的

#### 相关知识
全局变量和静态变量，在整个程序执行期间存在，局部变量存活于定义到对应的代码块执行结束。

全局静态变量文件内部共享，外部文件不能访问！

局部静态变量，函数内共享，函数外不能访问！
